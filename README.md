# 重要日期提醒服务

> 本服务当前主要是为了记录重要日期，进行提醒，当前已实现功能为生日提醒、纪念日提醒、倒计时提醒

| 模块                      | 描述                            |
|-------------------------|-------------------------------|
| zuji-remind-push-common | 该模块为公共模块，提供一些列公共方法            |
| zuji-remind-push-biz    | 该模块为业务实现模块，读取提醒任务，进行推送消息，具体实现 |
